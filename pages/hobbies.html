HOBBY 01: Programming

From learning how to code this simple website to more ambitious projects, I like to learn about programming.

My latest project is a little video game I made with a friend in 72h for the Ludum dare 46 (Game Jam Competition).



You can visit the game page (Click Here) or play/download (Click Here) on your browser.

The theme of the Game Jam was : "Keep it alive".
We decided to make a little puzzle game where the player have to place different tiles that will affect the movement of a rolling pangolin in order to reach the finnish line and prevent it from falling.
